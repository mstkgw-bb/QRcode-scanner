# 試薬スキャナー

スマホのカメラで試薬棚を映すだけで、目的の試薬をQRコードから素早く発見するWebアプリです。

## デモ

> GitHub Pagesを有効にすると `https://<ユーザー名>.github.io/<リポジトリ名>/` でアクセスできます。

## 機能

- カメラ映像からQRコードをリアルタイム検出
- 目的のQRコードIDを複数登録して一括検索
- 見つかったQRコードを緑枠でハイライト＋ラベル表示
- 発見時にバイブレーション通知
- 検出履歴の表示

## 使い方

1. 探したい試薬のQRコードIDを入力して「追加」
2. 複数の試薬を同時登録することも可能
3. 「カメラ起動」ボタンを押してカメラを向ける
4. 目的の試薬が見つかると緑の枠とラベルで強調表示される

## セットアップ（GitHub Pages）

1. このリポジトリをフォークまたはクローン
2. GitHub のリポジトリ設定 → **Pages** → Source を `main` ブランチのルートに設定
3. 数秒後に公開URLが発行される

## 技術スタック

- 純粋なHTML / CSS / JavaScript（フレームワーク不要）
- [jsQR](https://github.com/cozmo/jsQR) — QRコード検出ライブラリ（CDN読み込み）
- WebRTC `getUserMedia` API — カメラアクセス

## 注意事項

- カメラ機能にはHTTPS環境が必要です（GitHub Pagesは対応済み）
- 動作確認済みブラウザ: iOS Safari / Android Chrome / Desktop Chrome・Edge
- QRコードは蓋の上面に貼ると真上から映せて検出精度が上がります
